# Gerenciador de Tarefas
- Projeto em Python de um gerenciador de tarefas no terminal com menu interativo.

---

### Funcionalidades
- Adicionar tarefas
- Visualizar tarefas cadastradas
- Remover tarefas pelo número
- Validar entradas inválidas
- Encerrar o programa

---

### Conceitos praticados
- Listas
- Loops com `while`
- Condicionais
- Funções
- `try-except`
- Validação de entrada
- Organização com `main()`

---

### Como executar
1. Execute o arquivo `main.py`
2. Escolha uma opção do menu
3. Adicione, visualize ou remova tarefas
4. Use a opção 4 para sair
